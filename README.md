# Paqiang

Paqiang (mandarim: 爬墙; pinyin: páqiáng; IPA: /pʰa˧˥ tɕʰjaŋ˧˥/; significado: subir uma parede) é um conjunto de notas feitas em [Typst](https://typst.app) elaboradas para meu próprio aprendizado do mandarim (普通话).
